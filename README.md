# 💫 About Me:

Devops & SRE at **Mastercard**. I work on Kubernetes, observability (Loki, Prometheus, Grafana), CI/CD (Jenkins), and caching infrastructure (Valkey/Redis). 

AWS Community Builder (2025, 2026) and Docker Captain. Organizer of the [Valkey User Group Delhi/NCR](https://community.cncf.io/) (CNCF-listed). I write at [bloodraven.in](https://bloodraven.in) and speak at cloud-native meetups and conferences.

Open source contributor to [valkey-io/valkey-helm](https://github.com/valkey-io/valkey-helm).
